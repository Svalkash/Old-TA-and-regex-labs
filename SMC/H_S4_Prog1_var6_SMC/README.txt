Работать в папке /Debug
Запускать с файла autoLaunch.bat
Можно изменить в блокноте, поменять имя входного файла.
Вход: stringList.txt
Выход:
journal.txt - статистика серверов
log.txt - результаты по строкам

Весь код в /H_S4....
Main.cpp - запускаемый
ReaderClass.sm - для SMC
ReaderClass.h/.cpp - класс.
ServTable - контейнер.
комментариев по максимуму.

SmcLaunch - запускает SMC. Не нажимать - испортит _sm файлы.
